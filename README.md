## 🐾 About

**Ammal Farm** is a premium livestock and poultry farm located in Vellore, Tamil Nadu. This web application serves as the farm's digital storefront, allowing customers to browse elite breeds, view a curated gallery of farm life, and connect directly with the farm owners for inquiries.

### Key Features
- **Dynamic Gallery**: Real-time gallery powered by Firebase, allowing admins to highlight specific photos and videos.
- **Livestock Catalog**: Showcase of premium breeds including Nellore Judipi and Salem Black goats.
- **Admin Dashboard**: Secure interface for managing site content, uploading media, and updating descriptions.
- **SEO Optimized**: Fully configured with `sitemap.xml`, `robots.txt`, and Open Graph tags for better visibility on Google and social media.
- **Performance Monitoring**: Integrated with Vercel Analytics and Speed Insights.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18+)
- Firebase Account (for database and storage)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Madesh1205/Ammal-Farm.git
   cd Ammal-Farm
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Environment Setup**
   Create a `.env.local` file and add your Firebase configurations (refer to `.env.example`).

4. **Start Development Server**
   ```bash
   npm run dev
   ```
